========================================
COMMON QUESTIONS
========================================

Q: Can multiple people use it at once?
A: Yes! Streamlit handles multiple users automatically.

Q: Where are the files saved?
A: Nowhere! Users download a ZIP file directly.
   No files stored on your computer.

Q: Does it work on phone?
A: Yes! Works on any device with a browser.

Q: Can I password-protect it?
A: Yes, I can add authentication if you need it.

Q: What if I update the code?
A: Stop the app (Ctrl+C) and run it again.
   All users will see the new version immediately.
