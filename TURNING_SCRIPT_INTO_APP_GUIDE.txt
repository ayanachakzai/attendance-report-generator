HOW TO SET UP:
--------------

STEP 1: Install Streamlit

conda activate base
pip install streamlit


STEP 2: Save the App

Save attendance_app.py to your Downloads folder


STEP 3: Run the App

cd ~/Downloads
streamlit run attendance_app.py


STEP 4: Use the App

Your browser will open automatically to: http://localhost:8501

The app will have:
- File upload areas (drag & drop Excel and template)
- Dropdown to choose PDF or DOCX
- Checkbox to organize by groups
- Generate button
- Download button for ZIP file

========================================
CUSTOMIZATION OPTIONS
========================================

You can easily customize:

1. COLORS/THEME
   Create .streamlit/config.toml:
   [theme]
   primaryColor = "#FF4B4B"
   backgroundColor = "#FFFFFF"
   secondaryBackgroundColor = "#F0F2F6"

2. LOGO
   Add your college logo at the top:
   st.image("logo.png")

3. AUTHENTICATION
   Add login page so only authorized users access it

4. EMAIL NOTIFICATIONS
   Send email when reports are generated

5. DATABASE
   Save history of who generated what reports
